07

Wiring a chart in PI

One tool call

// In your PI session:
await pi.callTool("lumen-generate_visual", {
  type:    "chart",
  title:   "Q1 funnel",
  content: {
    chart: "bar",
    series: [{
      name: "signups",
      data: [120, 280, 410, 360],
    }],
    categories: ["Jan", "Feb", "Mar", "Apr"],
  },
  aesthetic: "dark-professional",
});
// → opens a self-contained HTML chart in your browser
